### 用户

1. 新用户注册, 

  a. 可以直接电话号码注册，用户名和密码发送到手机，每次注册都可以发送短信求登陆密码；用户注册成功后强制用户修改密码；用户名也需要填 不过可选； 电话号码是必填的唯一一项
  
  
2. 用户角色

  角色分为超级管理员，订单管理员
  
  
3. 会员用户列表

  会员列表可以操作用户：编辑/黑名单/删除

4. 管理员列表

  管理员列表可以操作用户：编辑/禁用/删除
  
5. 会员的地址管理
  

5. 用户登陆


### 订单

1. 订单列表

  订单列表需要有过滤工具，

2. 订单详情/订单状态跟踪


3. 订单日报/月报/周报统计报表


### 菜单

1. 菜单添加

2. 菜单列表

  菜单下市/菜单删除/菜单图片/查找
  
3. 菜单编辑/添加
